Some important questions and answer

Q1:Diferrence between var,let and const?

Ans: Var-The scope of a var variable is functional scope.It can be updated and re-declared into the scope. Let-The scope of a let variable is block scope.It can be updated but cannot be re-declared into the scope. Const-The scope of a const variable is block scope.It cannot be updated or re-declared into the scope.

Q2:Difference between map,filter,find and forEach

Ans: Map-It loops through every elements in an array and gives an array. forEach-It is as same as Map.Except is dosen't return by itself. filter-It works with an condition.It returns all the element whoever filled the condition. find-As same as filter except it returns the first element.

Q3:Difference between Arrow and regular function.

Ans:Unlike regular functions, arrow functions do not have their own this . The value of this inside an arrow function remains the same throughout the lifecycle of the function and is always bound to the value of this in the closest non-arrow parent function.
